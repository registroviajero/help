# Qué es RegistroViajero

RegistroViajero.com es una plataforma online que simplifica el cumplimiento del **Real Decreto 933/2021** para alojamientos turísticos en España. Permite recopilar los datos de los huéspedes, verificar sus documentos de identidad y enviar las comunicaciones obligatorias al Ministerio del Interior de forma automática.

## Cómo funciona

### 1. Registra tus alojamientos

Añade cada uno de tus alojamientos con su dirección, tipo (apartamento, hotel, casa rural, etc.) y código de establecimiento asignado por el Ministerio. Puedes gestionar múltiples alojamientos desde una misma cuenta de agencia.

### 2. Crea reservas

Cuando recibes una reserva, créala indicando las fechas de entrada y salida. También puedes **importar reservas automáticamente** desde plataformas como Booking.com, Airbnb o VRBO conectando el [calendario iCal](/guia/ical) de cada plataforma.

### 3. Envía el enlace de check-in

Cada huésped recibe un [enlace único](/guia/check-in). No necesitan registrarse ni instalar nada — abren el enlace en su móvil o navegador y completan sus datos.

### 4. Revisa y valida

Cuando todos los huéspedes han completado sus datos, revisa la información y valida la reserva.

### 5. Envía al Ministerio

Con un clic, RegistroViajero genera los documentos XML, los comprime y los [envía al sistema SES.HOSPEDAJES](/guia/enviar) del Ministerio del Interior.

## Requisitos para empezar

1. **Datos de tu agencia:** nombre comercial y CIF/NIF
2. **Datos de tus alojamientos:** dirección, tipo, código de establecimiento
3. **Credenciales SES.HOSPEDAJES:** usuario, contraseña y código de arrendador proporcionados por el Ministerio
4. **Navegador web** — Chrome, Safari, Firefox o Edge (versiones actuales)

::: tip
Si aún no tienes las credenciales del Ministerio, puedes empezar a usar RegistroViajero para gestionar reservas y recopilar datos de huéspedes. El envío al Ministerio se puede activar más adelante.
:::

## Limitaciones actuales

### Comunidades autónomas con policía propia

Actualmente RegistroViajero envía los datos al sistema **SES.HOSPEDAJES del Ministerio del Interior**, que cubre la mayoría de comunidades autónomas.

**No está disponible aún** la integración con:

- **Cataluña** (Mossos d'Esquadra) — Barcelona, Girona, Lleida, Tarragona
- **País Vasco** (Ertzaintza) — Álava, Guipúzcoa, Vizcaya

Estas comunidades utilizan sus propios sistemas. La integración está prevista para futuras versiones.

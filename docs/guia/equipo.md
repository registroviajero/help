# Equipo y roles

RegistroViajero permite trabajar en equipo con diferentes niveles de acceso.

## Roles disponibles

| Rol | Alojamientos | Reservas | Equipo | Configuración | Facturación | Auditoría |
|-----|:---:|:---:|:---:|:---:|:---:|:---:|
| **Propietario** | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |
| **Administrador** | ✓ | ✓ | ✓ | ✓ | — | ✓ |
| **Miembro** | — | ✓ | — | — | — | ✓ |

- **Propietario** — acceso completo, incluyendo facturación y gestión del equipo
- **Administrador** — gestión de alojamientos, reservas, equipo y configuración (sin facturación)
- **Miembro** — gestión de reservas y consulta de auditoría

## Invitar miembros

1. Ve a **Configuración → Equipo**
2. Haz clic en **Invitar miembro**
3. Introduce el email y selecciona el rol
4. El invitado recibirá un email con un enlace para crear su cuenta

::: info
Las invitaciones caducan a los 7 días si no se aceptan. Puedes reenviar la invitación o cancelarla en cualquier momento.
:::

## Gestionar el equipo

Desde la sección de equipo puedes:
- Cambiar el rol de un miembro
- Eliminar un miembro del equipo
- Ver las invitaciones pendientes

# Equipo y roles

RegistroViajero permite trabajar en equipo con diferentes niveles de acceso.

::: tip Equipo ilimitado
**Los miembros del equipo son ilimitados** — no pagas por persona. La factura se calcula solo por el número de alojamientos activos. Añadir compañeros es gratis.
:::

## Roles disponibles

| Rol | Alojamientos | Reservas | Credenciales SES | Equipo | Configuración | Facturación | Auditoría |
|-----|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| **Propietario** | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |
| **Administrador** | ✓ | ✓ | ✓ | ✓ | ✓ | — | ✓ |
| **Miembro** | — | ✓ | — | — | — | — | ✓ |

- **Propietario** — acceso completo, incluyendo facturación y gestión del equipo. Hay un único Propietario por agencia.
- **Administrador** — gestión de alojamientos, reservas, credenciales SES, equipo y configuración. No accede a la facturación.
- **Miembro** — gestión de reservas y consulta del registro de actividad. No puede editar alojamientos ni credenciales.

## Invitar miembros

1. Ve a **Configuración → Equipo**.
2. Haz clic en **Invitar miembro**.
3. Introduce el email y selecciona el rol.
4. El invitado recibirá un email con un enlace para crear su cuenta.

::: info
Las invitaciones caducan a los 7 días si no se aceptan. Puedes reenviar la invitación o cancelarla en cualquier momento.
:::

## Gestionar el equipo

Desde la sección de equipo puedes:

- Cambiar el rol de un miembro.
- Eliminar un miembro del equipo.
- Ver las invitaciones pendientes y reenviarlas o cancelarlas.

## Cambiar de Propietario

Si el Propietario actual deja la empresa, abre una solicitud de soporte: validamos el cambio y transferimos el rol. La transferencia se registra en el [registro de actividad](/guia/registro-actividad).

# Create reservations

## Create a manual reservation

1. Go to **Reservations** or the **Dashboard**.
2. Click **New reservation**.
3. Select the accommodation and the check-in and check-out dates.
4. The system automatically checks for overlaps with other reservations.

The reservation is created in the **Pending** state. Next, [add guests](/en/guide/check-in) and send the check-in links.

## Import reservations

You can import reservations automatically from booking platforms via [iCal calendars](/en/guide/ical). Imported reservations show a badge indicating their origin (Booking.com, Airbnb, VRBO, etc.).

Imported reservations without guest data are created empty — you decide whether to add guests or [block the dates](#blocking-dates).

## Blocking dates

When a reservation has no guests, you can mark it as **Blocked** to indicate those dates are unavailable (maintenance, personal use, etc.). You can unblock it at any time.

## Search

Use the search bar to find reservations by:

- Guest name
- Document number
- Reservation reference

## Export

- **PDF** — download a single reservation with all guest data.
- **CSV** — export all reservations filtered by state, property, and date range.

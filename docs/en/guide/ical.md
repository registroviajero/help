# Import calendars (iCal)

Connect the calendars from your booking platforms to import reservations automatically.

## Supported platforms

- Booking.com
- Airbnb
- VRBO
- Expedia
- Tripadvisor
- Google Calendar
- Any platform that exports iCal calendars (.ics)

## Connect a calendar

1. Open your accommodation in RegistroViajero.
2. In the **Calendar feeds** section, click **Add feed**.
3. Select the platform and paste the iCal export URL.
4. The system validates the connection immediately.

::: tip
You can connect multiple platforms to the same accommodation. Each feed syncs independently.
:::

## Synchronization

- **Automatic** every 15 minutes.
- **Manual** at any time from the feeds section.

## Imported reservations

Imported reservations appear with a badge indicating their origin (Booking.com, Airbnb, etc.).

Platforms that don't include guest data in the calendar (like Booking.com) create reservations without guests. From there you can:

- **Add guests** and send the check-in links.
- **Block the dates** if you don't need a guest registration.

## Where to find the iCal URL

### Booking.com
Extranet → Property → Calendar sync → Export calendar

### Airbnb
Calendar → Availability → Export calendar → Copy link

### VRBO
Calendar → Import/Export → Export calendar

# Install the app

RegistroViajero runs as a web application, but you can also **install it on your phone or tablet** to keep it on the home screen, launch it full-screen, and receive operating-system notifications.

::: info Mobile only
Installation is only available for the admin panel on mobile devices. Guest check-in works perfectly in any browser without installing anything.
:::

## Android (Chrome or Edge)

1. Open [app.registroviajero.com](https://app.registroviajero.com) on your phone.
2. Sign in.
3. You'll see a banner that says **Install RegistroViajero**. Tap **Install**.
4. If you don't see the banner, open the browser menu (⋮) and choose **Install app** or **Add to Home Screen**.

The app appears on your home screen and launches as a standalone application from the browser.

## iPhone / iPad (Safari)

iOS doesn't show an automatic banner. To install:

1. Open [app.registroviajero.com](https://app.registroviajero.com) in **Safari** (not Chrome).
2. Sign in.
3. Tap the **Share** icon (square with an upward arrow).
4. Scroll and pick **Add to Home Screen**.
5. Confirm the name and tap **Add**.

::: tip Push on iOS
On iPhone/iPad, operating-system push notifications **only work** if you have installed the app from the home screen. If you only access via Safari, you won't get push.
:::

## Push notifications

Once installed, go to **Settings → Notifications** and enable the alerts you want to receive as operating-system push. More detail in [Notifications](/en/guide/notifications).

::: warning Brave blocks push
If you use **Brave** as your browser, push notifications are blocked by its privacy restrictions. Install the app in Chrome or Edge.
:::

## Working offline

If you lose your Internet connection:

- RegistroViajero detects the disconnection and shows a banner at the top.
- Screens you have already visited remain available in read-only mode.
- As soon as the connection returns, RegistroViajero syncs automatically.

::: info Not offline-first
**Actions that require saving data** (create reservations, validate, submit to the Ministry) need a connection. RegistroViajero does not queue pending changes to send when the network returns.
:::

## How to uninstall

- **Android:** long-press the icon → **Uninstall**.
- **iPhone/iPad:** long-press the icon → **Remove app** → **Remove from Home Screen** (does not delete data).

You can keep accessing [app.registroviajero.com](https://app.registroviajero.com) from your browser as before.
